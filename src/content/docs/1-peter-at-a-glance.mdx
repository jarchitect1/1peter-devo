## 1. 写作背景

| | |
|---|---|
| **作者** | 使徒**彼得**（1:1），由西拉（Silvanus）代笔书写（5:12）。早期教会普遍接受；彼得作者身份是福音派的标准立场。 |
| **收信人** | 分布在本都、加拉太、加帕多家、亚西亚、庇推尼的"**分散寄居的选民**"（1:1），主要是小亚细亚北部的外邦信徒。 |
| **写作地点** | "巴比伦"（5:13），几乎可以肯定是**罗马**的象征性名称，约主后 62–64 年，尼禄迫害达到顶峰之前。 |
| **写作缘由** | 信徒面对社会的敌视、毁谤，并"为这名"受试炼。彼得坚定他们的盼望，呼召他们过圣洁生活，并教导他们按基督的样式善度受苦。 |
| **核心比喻** | **客旅与寄居者**（1:1、17；2:11）， 天上的国民，在一个注视的世界中作恩典的侨民。 |
| **基调** | 温柔而迫切，充满赞美与牧者情怀，出自一位亲身经历受苦与失败的牧者之手。 |

## 3. 一句话总览

<KeyIdea label="一息之间读懂本书">
因为神在复活的基督里赐给我们活泼的盼望，并使我们成为祂所拣选、圣洁的祭司，所以我们是忠心的寄居者，品行圣洁，在各等本分中顺服，受苦中喜乐，教会中谦卑，警醒直到末了。
</KeyIdea>

## 4. 贯穿五章的主线

1. **身份先于行为**：我们是谁（蒙拣选的寄居者、重生之人、君尊的祭司），是我们当如何行的根基。
2. **复活所确保的盼望**：存留在天上、不能朽坏的基业，如今活泼而有功效（1:3-5）。
3. **作神子民的圣洁**："你们要圣洁，因为我是圣洁的"（1:15-16），并在世上活出来。
4. **客旅与寄居者**：以天家为家，在地上寄居却不安于世（1:1、17；2:11）。
5. **跟随基督受苦**：按牧人的样式忍耐承受不义的苦，并带着祝福的心志（2:18-25；3:13-22；4:12-19）。
6. **在神的秩序中顺服**：顺服权柄、主人、婚姻之内，以及羊群之内（2:13-3:7；5:5）。
7. **恩典、谦卑、警醒**：祭司直到末了当有的姿态（5:5-11）。

<Aside title="本系列的主线" type="note">
**信徒皆祭司**（2:9）。每一课都可追溯至此：一个圣洁的子民献上属灵的祭物，宣扬神的美德，彼此牧养与服侍，直到受苦让位于荣耀。
</Aside>
