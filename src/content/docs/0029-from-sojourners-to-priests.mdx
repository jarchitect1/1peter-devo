  <Scripture ref="彼得前书 1:1 (和合本)">
<sup class="vnum">1</sup>耶稣基督的使徒彼得，写信给那分散在本都、加拉太、加帕多家、亚细亚、庇推尼寄居的，
</Scripture>

  <Scripture ref="彼得前书 2:9 (和合本)">
<sup class="vnum">9</sup>惟有你们是被拣选的族类，是有君尊的祭司，是圣洁的国度，是属神的子民，要叫你们宣扬那召你们出黑暗、入奇妙光明者的美德。
</Scripture>

  <p class="lead opener">这是三课结束综论中的第一课，从逐节行走退一步，追踪书信的大主线。我们从系列开始之处开始：身份。彼得开篇命名他的读者为<em>蒙拣选的寄居者</em>（1:1），并登顶命名他们为<em>有君尊的祭司</em>（2:9）。整封书信活在两个名字之间。彼得所命令的一切，都从他首先说我们是谁涌流而出。追踪身份主线，书信的行为就连贯成一种一致的生命，一个被作成祭司的寄居者的生命。</p>

  ## <span class="num">1.</span> 框定书信的两个名字

  <p>在 1:1，第一个名字：<strong>蒙拣选的寄居者</strong>。已被拣选，已离家。信徒的首要身份是双重的，扎根于永恒（蒙拣选），活出于流离（寄居者）。<a href="/glossary/">寄居者</a>一词成为全书标志，在 1:17 与 2:11 重现。在彼得告诉他们当怎样活之前，他告诉他们当在哪里活：不在家中。</p>

  <p>在 2:9，第二个名字，系列的贯穿主线：<strong>被拣选的族类，有君尊的祭司，圣洁的国度，属神的子民</strong>。寄居者成了圣殿。分散者成了子民。这四重名字取自出 19:5-6，那里神在西奈命名以色列为属乎自己，而彼得毫不抱歉地放在一个以外邦人为主的教会身上。寄居者被作成了祭司。</p>

  <Aside type="note" title="请留意">
1:1 与 2:9 之间的距离，正是书信所行的距离：从<em>寄居者</em>到<em>祭司</em>。然而这两个名字并非对立。祭司仍是寄居者（2:11）；寄居者已是祭司（2:5）。信徒同时持守两者：离家，并在圣殿中事奉。两种身份都承重直到书信末尾。
</Aside>

  ## <span class="num">2.</span> 身份在行为之先，贯穿始终

  <p>请看这模式如何贯穿全书。彼得每次都先<em>命名</em>再<em>命令</em>。</p>

  <ul>
    <li><strong>1:3-12</strong>（重生、被保守、被炼净）在<strong>1:13-16</strong>（所以，约束心思，要圣洁）之前。</li>
    <li><strong>1:18-19</strong>（你被宝血赎回）在<strong>1:17</strong>（当存敬畏度日）之前。</li>
    <li><strong>2:9-10</strong>（你是有君尊的祭司）在<strong>2:11-12</strong>（作为寄居者，禁戒；保持品行端正）之前。</li>
    <li><strong>2:24</strong>（因祂鞭伤你已得医治）在<strong>2:21</strong>（跟随祂脚踪）之前。</li>
    <li><strong>5:10</strong>（赐诸般恩典的神要恢复你）包裹<strong>5:6-9</strong>的命令（自卑、卸、抵挡）。</li>
  </ul>

  <p>Adrian Rogers 抓住这牧养逻辑：神从不要求信徒去生产恩典所未先提供的（<a href="https://www.lwf.org/">Rogers 论彼前 1:13</a>）。恩典赐身份，身份承担行为。有君尊的祭司不是凭行为进入祭司职分；他<em>因为</em>是祭司而如此行事。</p>

  <Aside type="note" title="灵修要点">
书信的工作原则是身份在行为之先。你不是赚得祭司职分；你是领受它，然后活出它。若你的顺服失去把握，先当看的不是你的意志力，而是你的身份。你忘了你是属谁的？
</Aside>

  ## <span class="num">3.</span> 服事、顺服、受苦、牧养的祭司群体

  <p>一旦身份定下，整封书信的行为就落入祭司生命自然形状的位置。有君尊的祭司群体是：</p>

  <ul>
    <li><strong>服事的祭司群体</strong>（2:5；4:10-11），献上灵祭，使用所受的恩赐。</li>
    <li><strong>顺服的祭司群体</strong>（2:13-3:7；5:5），"为主的缘故"安排自己服于公民权柄、职场、婚姻、羊群之下。</li>
    <li><strong>受苦的祭司群体</strong>（2:18-25；3:13-22；4:12-19），在留下榜样之代替者的模式后，承受不义的对待。</li>
    <li><strong>牧养与谦卑的祭司群体</strong>（5:1-7），由甘心服事的长老带领，以谦卑束腰，把忧虑卸给顾念人的神。</li>
    <li><strong>警醒的祭司群体</strong>（5:8-11），警醒防狮子，以坚固信心抵挡，止于恢复的颂赞。</li>
  </ul>

  <p>请注意这些无一是一项节目。它们是一个身份自然的流出。祭司服事，因为他是祭司。他顺服，因为他的主顺服。他受苦，因为他的救主受苦。他牧养，因为大牧者告诉他要喂养羊。他警醒，因为主人告诉他要保持清醒。一个身份，五个重叠的形状，一个一致的生命。</p>

  ## <span class="num">4.</span> 寄居者-祭司，宣扬祂的美德

  <p>而 2:9 的目的子句是把这一切系在一起的职分说明：<strong>"要叫你们宣扬那召你们出黑暗、入奇妙光明者的美德"</strong>（2:9）。寄居者-祭司为宣扬而存在。宣扬借言语（3:15，盼望的缘由）与生命（2:12，被看见的好行为）。整封书信在某种意义上，是一本有君尊祭司群体如何在观看、敌意的世界中宣扬其神之美德的手册。</p>

  <p>所以身份主线绕一圈回来。在 1:1 我们是<em>蒙拣选的寄居者</em>，离家在外。在 2:9 我们是<em>有君尊的祭司</em>，被赐目的。而在 5:10，那赐诸般恩典、召我们到祂永远荣耀里的神，要亲自成全、坚固、赐力量、建立我们。寄居者抵家，祭司被平反，身份永远封定。这就是身份主线的终点。</p>

  <KeyIdea label="一句话核心">
整封书信活在两个名字之间：<em>蒙拣选的寄居者</em>（1:1）与<em>有君尊的祭司</em>（2:9）。每条命令都从那身份涌流，绝非相反。祭司服事、顺服、受苦、牧养、警醒，全为宣扬那召他出黑暗之 神的美德。活出身份，行为随之而来。
</KeyIdea>

  <Aside type="note" title="试一试">
把这两个名字写在你的一周之上：<em>我是蒙拣选的寄居者，我是有君尊的祭司。</em>当一个决定或试探来到，问：<em>寄居者-祭司在此当怎样行？</em>让身份承担行为，而非相反。
</Aside>

  ## 应用：头、心、手

  <p><strong>头。</strong>相信你的身份在你被要求行为之先已赐下，你是蒙拣选的寄居者与有君尊的祭司，而书信中每条命令都从那身份涌流。恩典赐予；身份承担；行为跟随。</p>

  <p><strong>心。</strong>培养你在基督里是谁的稳固意识。治死忘记身份而试图赚取的失忆，也治死把处境读作对你地位判决的绝望。寄居者已是祭司。</p>

  <p><strong>手。</strong>本周从一个身份、而非朝向身份，活出一个决定。对一个真实选择问：<em>寄居者-祭司在此当怎样行？</em>按情况所需服事、顺服、受苦、牧养或警醒，以宣扬祂的美德为目标。</p>
