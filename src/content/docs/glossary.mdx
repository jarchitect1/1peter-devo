  <p class="passage">开启本书信的词汇</p>

<p class="lead">一份简短的词汇表。当这些词融入你的骨髓，全书便读起来如同一脉相承的论点：一个圣洁的祭司，活出活泼的盼望。</p>

<dl class="glossary">

<dt>Parepidēmos / paroikos <span class="grk">παρεπίδημος / πάροικος</span></dt>
<dd>
  <em>寄居者、客旅、外侨。</em>彼得对信徒的标志性称呼：不是游客，而是不属此地的居民。地上不是家，天上才是。这一姿态塑造了圣洁、盼望，以及我们如何承受毁谤。
  <span class="ref">关键经文：彼前 1:1、17；2:11。</span>
</dd>

<dt>Elpis <span class="grk">ἐλπίς</span></dt>
<dd>
  <em>盼望</em>，却非一厢情愿的遐想，而是锚定于基督复活与存留在天之基业的笃定而活泼的期待。"专心仰望……的恩典"（1:13）。
  <span class="ref">关键经文：彼前 1:3、13、21；3:15。</span>
</dd>

<dt>Anagennaō / anagennēsis <span class="grk">ἀναγεννάω</span></dt>
<dd>
  <em>重生、使再次出生。</em>借着神活泼常存的道而有的新生命，"得着活泼的盼望"（1:3）。不能朽坏的种子，不能朽坏的生命。
  <span class="ref">关键经文：彼前 1:3、23。</span>
</dd>

<dt>Ekteneōs <span class="grk">ἐκτενῶς</span></dt>
<dd>
  <em>切实地、竭力地、全力伸张地。</em>彼得所命令的那种爱：不是客套的温情，而是用力伸展、竭力去爱的爱，能"遮掩许多的罪"（4:8）。同一个词用于路加福音 22:44 中主耶稣恳切的祷告。教会试炼中的爱当有这种全力、不懈的特质。
  <span class="ref">关键经文：彼前 1:22；4:8。</span>
</dd>

<dt>Hagios <span class="grk">ἅγιος</span></dt>
<dd>
  <em>圣洁、分别出来。</em>"那召你们的既是圣洁，你们在一切所行的事上也要圣洁，因为经上记着说：'你们要圣洁，因为我是圣洁的。'"圣洁是神子民的家庭特征。
  <span class="ref">关键经文：彼前 1:15-16；2:5、9。</span>
</dd>

<dt>Basileion hierateuma <span class="grk">βασίλειον ἱεράτευμα</span></dt>
<dd>
  <em>有君尊的祭司。</em>本系列的主线。每位信徒都是祭司：借着基督直接进到神面前，并有献上属灵祭物、宣扬神美德的呼召。没有独立的祭司阶层。
  <span class="ref">关键经文：彼前 2:5、9。</span>
</dd>

<dt>Laos eis peripoiēsin <span class="grk">λαὸς εἰς περιποίησιν</span></dt>
<dd>
  <em>属神的子民。</em>取自出埃及记 19:5-6，如今应用于教会，犹太人与外族人合而为一。神所珍视的产业，为彰显祂的荣耀而被建立。
  <span class="ref">关键经文：彼前 2:9-10。</span>
</dd>

<dt>Oikodomeō / oikos pneumatikos <span class="grk">οἰκοδομέω / οἶκος πνευματικός</span></dt>
<dd>
  <em>建造 / 灵宫。</em>基督是活石，信徒是活石，被建造成为神借圣灵居住的殿。先有群体的属性，才有个人。
  <span class="ref">关键经文：彼前 2:4-5。</span>
</dd>

<dt>Paschō / pathēma <span class="grk">πάσχω / πάθημα</span></dt>
<dd>
  <em>受苦 / 苦难。</em>不是绕道，而是呼召：信徒被"蒙召"如此，因为基督为他们受苦，留下榜样（2:21）。受苦试验、炼净，并参与基督通往荣耀的道路。
  <span class="ref">关键经文：彼前 2:19-21；3:14、17；4:1、12-13、15-16、19；5:9-10。</span>
</dd>

<dt>Hypotassō <span class="grk">ὑποτάσσω</span></dt>
<dd>
  <em>顺服、把自己置于其下。</em>自愿将自己安放于神所设立的结构中（公民权柄、主人、婚姻、羊群），如同归于主。顺服并非低人一等，而是对神主权的信靠。
  <span class="ref">关键经文：彼前 2:13、18；3:1、5；5:5。</span>
</dd>

<dt>Anastrephō / anastrophē <span class="grk">ἀναστρέφω / ἀναστροφή</span></dt>
<dd>
  <em>行事为人 / 生活方式。</em>彼得反复呼吁的一种<em>可见的</em>圣洁：以"端正的品行"使无知闭口，并为福音作美好的见证。
  <span class="ref">关键经文：彼前 1:15、18；2:12；3:1-2、16。</span>
</dd>

<dt>Tapheinophrosynē / tapeinoō <span class="grk">ταπεινοφροσύνη / ταπεινόω</span></dt>
<dd>
  <em>心思谦卑 / 自己谦卑。</em>"你们众人都要彼此以谦卑束腰"（5:5）。祭司的衣裳，在神大能的手之下。
  <span class="ref">关键经文：彼前 3:8；5:5-6。</span>
</dd>

<dt>Poimainō / poimnion <span class="grk">ποιμαίνω / ποίμνιον</span></dt>
<dd>
  <em>牧养 / 羊群。</em>长老甘心、乐意、作榜样地牧养神的羊群，在那将要显现的牧长之下。是聚会中仆人式领袖的典范。
  <span class="ref">关键经文：彼前 5:2-4。</span>
</dd>

<dt>Grēgoreō / nēphō <span class="grk">γρηγορέω / νήφω</span></dt>
<dd>
  <em>警醒 / 谨守。</em>祭司末世当有的姿态：警醒抵挡那吼叫的魔鬼，在盼望中节制，用坚固的信心抵挡他。
  <span class="ref">关键经文：彼前 1:13；4:7；5:8。</span>
</dd>

<dt>Silvanus <span class="grk">Σιλουανός</span></dt>
<dd>
  西拉，彼得忠心的同工，本书信的代笔人（5:12），即曾与保罗同工的西拉（徒 15-18）。是彼得与保罗两个圈子之间默默的桥梁。
  <span class="ref">关键经文：彼前 5:12。</span>
</dd>

</dl>
